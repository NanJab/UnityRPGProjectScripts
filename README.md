## 📌 개요  
- **장르**: 액션 RPG  
- **개발 기간**: 2024.01 ~ 2024.06  
- **키워드**: `Unity` `C#` `인벤토리 시스템` `퀘스트`  

## 🛠 기능 구현  
| 기능 | 설명 |  
|------|------|  
| **인벤토리** | 장비 장착/해제, 아이템 정렬 |  
| **스킬 시스템** | 쿨타임 UI 연동 |  
| **몬스터 AI** | NavMesh 기반 추적 |
